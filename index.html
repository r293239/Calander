📋 Website Cloner

Fetch HTML/CSS/JS from any site and host it on GitHub Pages

Enter a URL and click "Fetch & Clone"

Download the cloned files to upload to your GitHub repository:

Then upload to GitHub → Settings → Pages to deploy

📌 How to use:
1. Enter a website URL (must be accessible from your network)
2. Click "Fetch & Clone" - the HTML will be captured
3. Download the files and upload to a new GitHub repository
4. Enable GitHub Pages in repository Settings
5. Access your cloned site at: username.github.io/repo-name
